Fitur Utama

Audit Logger menyediakan beragam fitur untuk mempermudah pencatatan dan analisis aktivitas pengguna pada aplikasi Anda.

REST API

API REST yang lengkap dengan dukungan filter dan pagination untuk mencatat dan mengambil log dengan mudah.

gRPC Service

Layanan gRPC yang efisien untuk integrasi dengan berbagai bahasa pemrograman dengan performa tinggi.

Export Data

Dukungan export data log ke format CSV dan JSON sesuai kebutuhan analisis dan pelaporan Anda.

Webhook Triggers

Dukungan webhook untuk notifikasi real-time saat terjadi perubahan penting di sistem Anda.

JWT Authentication

Keamanan API dengan JWT dan dukungan rate limiting untuk melindungi dari penyalahgunaan akses.

Admin Panel

Panel admin untuk melihat dan mengelola log dengan antarmuka yang mudah digunakan dan intuitif.

API Endpoints

Berikut adalah beberapa endpoint utama yang tersedia di Audit Logger API untuk kebutuhan integrasi Anda.

POST /api/log/

Mencatat log aktivitas baru dengan detail perubahan yang terjadi.

Create Authentication
GET /api/log/

Menampilkan daftar log dengan dukungan filter dan pagination yang lengkap.

Query Filter
GET /api/log/export/csv/

Export data log ke format CSV untuk kebutuhan analisis lebih lanjut.

Export CSV
GET /api/log/export/json/

Export data log ke format JSON untuk integrasi dengan sistem lain.

Export JSON
POST /api/auth/login/

Endpoint untuk login dan mendapatkan token JWT untuk autentikasi.

Auth JWT
